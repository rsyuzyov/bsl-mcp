Kiro / Supermaven

Добавьте в файл ~/.kiro/settings/mcp.json:

{
  "mcpServers": {
    "1c-rag": {
      "url": "http://localhost:8000/sse"
    }
  }
}

Claude Desktop

Добавьте в конфигурационный файл:

{
  "mcpServers": {
    "1c-rag": {
      "command": "python",
      "args": ["C:\\путь\\к\\repo\\bsl-mcp\\code-search.py"]
    }
  }
}

Примечание: Рекомендуется использовать SSE подключение (http://localhost:8000/sse), если клиент поддерживает его.
